# 判镜 JudgMirror

**判断工程化的开源参考实现。**

口号：看清楚你到底输在哪。

---

## 这是什么

判镜不是一个独立 AI。它寄生在 LLM 上——Claude、GPT、任何基础模型都行。

它做一件事：**把你的判断过程，从直觉变成可看见、可怀疑、可修正的结构。**

你继续用你习惯的 LLM 聊天。判镜在背后捕获决策点、拆维度、记录预测、等数据回来后做对照、告诉你你的判断偏在哪。

---

## 核心思想

LLM 可以执行。但"该不该执行""往哪个方向执行"——这是判断。

AGI 会吃掉执行层。判断层的人机差异在于：**谁有校准回环，谁就有判断力。**

判镜就是校准回环的工程化实现。
